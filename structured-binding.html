Structure Binding

Structed Binding 是如何工作的

在 C++ 17 中可以写出如下的代码：

std::string func() {
    auto [succeeded, value] = get_some_pair();
    return value;
}

编译器会把以上的代码翻译成如下的代码：

std::string func() {
    auto e = get_some_pair();
    auto& succeeded = e.first;
    auto& value = e.second;
    return value
}

由于 NRVO 不能用在 sub object 因此，从性能方面考虑，如下的代码相比性能较好：

std::string func() {
    auto [succeeded, value] = get_some_pair();
    return std::move(value);
}
